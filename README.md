# Task 1 Title: Data Cleaning

## Description
This tasks involves cleaning and preprocessing the Netflix TV Shows dataset to prepare it for analysis. The main goals are to handle missing data, remove duplicates, and ensure data consistency.

## Dataset
- Source: Netflix Movies and TV Shows dataset (Kaggle)
- Key columns: show_id, title, director, date_added, etc.

## Data Cleaning Steps
- Loaded data into Excel Power Query.
- Identified and replaced missing values in certain columns.
- Removed duplicate rows based on show_id and title.
- Standardized date formats and column headers.

## Tools Used
- Microsoft Excel Power Query

## Deliverables
- Cleaned dataset file ready for analysis.
- Documentation of cleaning steps.

## Usage
Open the cleaned dataset in Excel or any data analysis tool.
